About

The security alarm sensor's flow chart logic was implemented in simplified block code through user friendly software called nanoNavigator using a programmable logic controller (PLC). The main goal of the project was to create a device which can detect movement, detect potentially dangerous objects and inform the user when any of the above situations may occur.

Inputs peripherals:
I-00: Diffused photoelectric sensor
I-02: Inductive Sensor
I-03: Push Button

Output peripherals:
Q-00: Blue light
Q-01: Orange light
Q-02: Buzzer
